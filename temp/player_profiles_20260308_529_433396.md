# Marc Bernal

```text
生まれ::2007年5月26日、スペイン・ベルガ
国籍::スペイン
ポジション::守備的MF
身長・利き足::193cm / 左利き
経歴::2014-2024: バルセロナ（ラ・マシア）
2024-: バルセロナ
特徴::190cmを超える恵まれた体格と、卓越したパスセンスを併せ持つ「次世代のピボーテ」。左足一本でゲームを組み立てる姿は、クラブのレジェンドであるセルヒオ・ブスケツを彷彿とさせる。
監督・選手の発言::ハンス＝ディーター・フリック監督が就任後、最初に目をかけたラ・マシアの至宝。監督は「彼は17歳とは思えないほどピッチ上で成熟している」と賛辞を贈った。
記録::2024年8月のバレンシア戦にて、17歳84日でトップチームデビュー。これはリオネル・メッシをも上回るクラブ史上8番目の若さであり、フリック体制下でのラ・マシア出身者デビュー第1号となった。
面白いエピソード::チームメイト思いで知られ、パウ・クバルシやジェラール・マルティンとは大親友。寮生活時代からの絆が深く、ピッチ外でも行動を共にすることが多い。
意外な経歴::デビュー直後の2024年8月末に膝の前十字靭帯（ACL）を断裂するという悲劇に見舞われたが、クラブは彼を全面的に支持。リハビリ期間中にもかかわらず、2029年までの長期契約と5億ユーロの契約解除金を設定し、その将来を確約した。2025年9月に382日ぶりの復帰を果たし、復帰戦でいきなりアシストを記録した。
```
