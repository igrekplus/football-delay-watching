# Randal Kolo Muani

```text
生まれ::1998年12月5日、フランス・ボンディ
国籍::フランス
ポジション::センターフォワード / 右ウイング
身長・利き足::187cm / 右利き
経歴::2005-2010: ヴィルパントFC
2010-2015: トランブレFC
2015-2016: トルシー
2016-2022: ナント
2019-2020: ブローニュ（期限付き移籍）
2022-2023: フランクフルト
2023-2025: パリ・サンジェルマン
2025-: トッテナム（期限付き移籍）
特徴::長いストライドで背後へ抜ける速さと、前線でボールを収めて味方を走らせる連係力を兼ね備えています。ラインブレイク一辺倒ではなく、右へ流れて局面を作り直したり、カウンターの起点になったりできる万能型FWです。
監督・選手の発言::加入時のクラブ公式コミュニケーションでは、プレミアリーグで自分のスピードとダイナミズムを示したい意欲が前面に出ていました。最前線でもサイドでも使える柔軟性が、チーム事情と噛み合う補強として受け止められています。
記録::2022年ワールドカップでフランス代表の一員として決勝まで進出。ナントではクープ・ドゥ・フランス優勝、フランクフルトでは欧州カップ戦出場と得点力向上を経験し、代表とクラブの両方で階段を上がってきました。
面白いエピソード::ボンディ出身で、同じ地域から世界へ出たキリアン・エムバペと並べて語られることも多い選手です。ナントでの下積みが長く、ブローニュへのローンも経験してから一気に代表級へ駆け上がった遅咲き寄りの成功例でもあります。
意外な経歴::爆発的なブレイク前はナントBやローン先で地道に出場数を積み、いきなりスター扱いされた選手ではありませんでした。フランクフルトで一気に評価を跳ね上げ、PSGを経てトッテナムへたどり着いたジャンプアップ型のキャリアです。
```
