# Kevin Danso

```text
生まれ::1998年9月19日、オーストリア・フォイツベルク
国籍::オーストリア
ポジション::センターバック
身長・利き足::190cm / 右利き
経歴::2006-2014: レディング、MKドンズなどイングランドで育成
2014-2017: アウクスブルク（下部組織）
2016-2021: アウクスブルク
2019-2020: サウサンプトン（期限付き移籍）
2020-2021: フォルトゥナ・デュッセルドルフ（期限付き移籍）
2021-2025: ランス
2025-: トッテナム
特徴::対人守備の強さとリカバリーの速さを兼ね備えた右利きのセンターバックです。空中戦でも地上戦でも強度を出せるうえ、押し込まれた場面でも前に潰しに出られるため、ラインを高く保ちたいチームと相性が良いタイプです。
監督・選手の発言::加入時にはクラブ公式で、トッテナム行きを「大きなチャンス」と表現していました。イングランド育ちでもあるため、プレミアリーグ挑戦をキャリアの自然な次の段階として捉えていたことがうかがえます。
記録::2017年に10代でオーストリア代表デビューを果たし、複数年代の代表を経ずにA代表へ早くから食い込んだ有望株でした。ランス時代にはリーグ・アン屈指のデュエル勝率を誇る守備者として評価を高めています。
面白いエピソード::生まれはオーストリアですが、6歳でイングランドへ移り、MKドンズ周辺で育った経歴の持ち主です。大陸型CBのサイズ感と、イングランド育ちのフィジカル文化が同居しているのがキャリアの面白さです。
意外な経歴::サウサンプトン時代はプレミアリーグで定着できず遠回りも経験しましたが、ドイツとフランスで再構築してから再びイングランド上陸を果たしました。一直線のエリート街道ではなく、複数リーグを経て完成度を高めた再上陸型のDFです。
```
