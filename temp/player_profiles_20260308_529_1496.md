# Raphinha

```text
生まれ::1996年12月14日、ブラジル・ポルト・アレグレ
国籍::ブラジル / イタリア
ポジション::ウィング
身長・利き足::176cm / 左利き
経歴::2018-2019: スポルティングCP
2019-2020: レンヌ
2020-2022: リーズ
2022-: バルセロナ
特徴::圧倒的なテクニックとスピードを誇るブラジル代表の11番。2024-25シーズンはハンス＝ディーター・フリック監督の下で覚醒し、公式戦34ゴール22アシストという驚異的な記録をマーク。チャンピオンズリーグでも得点王に輝くなど、世界最高のウィングの一人としての地位を確立した。
監督・選手の発言::フリック監督は「ハフィーニャは戦士だ。攻撃だけでなく守備の戻りも一級品で、チームに欠かせないリーダーだ」と絶賛。2024-25シーズンのバロンドール投票でも5位にランクインした。
記録::2024-25シーズン チャンピオンズリーグ共同得点王（13ゴール）およびアシスト王。同シーズン、バルセロナで国内三冠（ラ・リーガ、コパ・デル・レイ、スーペルコパ）を達成。
面白いエピソード::幼少期はポルト・アレグレのファベーラ（貧民街）で育ち、空腹で食べ物を乞うこともあったという苦労人。憧れの選手はロナウジーニョで、7歳の時に彼の誕生日パーティーに招待された（父親が友人だったため）ことがサッカー選手を志すきっかけとなった。
意外な経歴::ブラジル代表デビュー戦（2021年ベネズエラ戦）では、途中出場からわずか数分で2アシストを記録し、鮮烈なデビューを飾った。
```
