# Eric García

```text
生まれ::2001年1月9日、スペイン・マルトレル
国籍::スペイン
ポジション::センターバック / 守備的MF
身長・利き足::182cm / 右利き
経歴::2008-2017: バルセロナ（ラ・マシア）
2017-2021: マンチェスター・シティ
2021-: バルセロナ
2023-2024: ジローナ（ローン）
移籍金::マンチェスター・シティからフリー移籍でバルセロナへ復帰。
特徴::卓越した足元の技術と戦術眼を持つビルドアップの要。冷静に相手のプレスを剥がし、前線へ正確な楔のパスを打ち込む。ハンス＝ディーター・フリック体制ではセンターバックだけでなく、ボランチとしても新境地を開拓した。
監督・選手の発言::グアルディオラ監督はシティ時代に「彼は非常に賢く、常に監督が何を求めているかを理解している。17歳とは思えない完成度だ」と絶賛。バルセロナのチーム内でもリーダーシップが高く評価されている。
記録::2024年パリオリンピックでスペイン代表の守備の柱として金メダルを獲得。2025年3月にはバルセロナでの通算100試合出場を達成した。
面白いエピソード::バルセロナのアカデミー時代からキャプテンを務めることが多く、そのプレースタイルとリーダーシップから「プジョルの心を持つピケ」と称されたこともある。
意外な経歴::名字の「ガルシア」はスペインで最も多い名字であり、彼はバルセロナの歴史の中で同姓を持つ23人目の選手である。
```
