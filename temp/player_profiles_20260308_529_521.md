# Robert Lewandowski

```text
生まれ::1988年8月21日、ポーランド・ワルシャワ
国籍::ポーランド
ポジション::センターフォワード
身長・利き足::185cm / 右利き
経歴::2010-2014: ドルトムント
2014-2022: バイエルン・ミュンヘン
2022-: バルセロナ
特徴::「点取り屋」という言葉を体現する、21世紀最高のストライカーの一人。35歳を超えても衰えを知らない得点能力を誇り、2024-25シーズンには欧州5大リーグで最も早く公式戦40ゴールに到達。チャンピオンズリーグ通算100ゴール達成という金字塔も打ち立てた。
監督・選手の発言::フリック監督とはバイエルン時代からの師弟関係。「彼はエリア内での世界最高の解決策。年齢はただの数字だということを証明し続けている」と評される。
記録::2024-25シーズン ラ・リーガ・ピチーチ（得点王）。CL通算100ゴール達成（史上3人目、ロナウドより12試合早いペース）。欧州5大リーグの異なる3クラブで100ゴール以上を記録した数少ない選手。
面白いエピソード::食事と睡眠へのこだわりは異常なほどで、消化を助けるためにデザートから食べ始め、次に主食、最後に前菜という「逆順」で食事をすることがある（Lewandowski dietとして有名）。
意外な経歴::父親は柔道のチャンピオン、母親と姉はプロのバレーボール選手、妻は空手の世界大会メダリストという超アスリート一家。
```
