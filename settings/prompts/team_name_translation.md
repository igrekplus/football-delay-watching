以下のサッカーチーム名を日本語（カタカナ）に変換してください。

ルール:
1. 一般的に使われるカタカナ表記を優先してください。
2. 「FC」「CF」「SC」等は省略可能です（例: "FC Barcelona" → "バルセロナ"）。
3. 複数の単語がある場合は、中黒（・）で区切ってください。
4. プレミアリーグ、ラ・リーガなどの主要リーグのチーム名は、日本のスポーツメディアで一般的に使われる表記に従ってください（例: "Manchester United" → "マンチェスター・ユナイテッド"）。

JSON形式で出力してください（余計な説明は不要です）:
出力JSON形式:
{{
  "チーム名（英語）": {{
    "katakana": "カタカナ表記",
    "keywords": ["略称1", "略称2", ...]
  }}
}}

例:
{{
  "Manchester City": {{
    "katakana": "マンチェスター・シティ",
    "keywords": ["マンチェスター・C", "マンC", "シティ"]
  }},
  "FC Barcelona": {{
    "katakana": "バルセロナ",
    "keywords": ["バルサ"]
  }},
  "Borussia Mönchengladbach": {{
    "katakana": "ボルシア・メンヒェングラートバッハ",
    "keywords": ["ボルシアMG", "グラートバッハ"]
  }}
}}
