ルール:
1. 日本人選手、または漢字圏の選手で一般的に漢字表記される場合は、漢字で表記してください（例: Takefusa Kubo → 久保建英）。この場合、短縮名は作成せずフルネームと同じにしてください。
2. 外国人選手はカタカナで表記し、一般的に使われる表記を優先してください（例: Mohamed Salah → モハメド・サラー）。
3. カタカナ表記の場合、姓名の間は中黒（・）で区切ってください。
4. **短縮名**: 外国人選手の場合、「ファーストネームの頭文字 + ・ + ラストネーム」としてください（例: Erling Haaland → E・ハーランド、Kevin De Bruyne → K・デ・ブライネ）。
