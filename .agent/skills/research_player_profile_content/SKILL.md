# 選手プロフィール本文調査スキル

## 概要

この skill は、選手プロフィールの**本文づくり**に特化します。

担当範囲は以下です。

1. 情報収集
2. `labelled_lines_v1` 形式への整形
3. セルフレビュー
4. `temp/player_profiles_YYYYMMDD_[team]_[player].md` 出力

GCS upload、standalone HTML 生成、deploy は親 skill 側で扱います。

## 情報源の優先順位

| 優先度 | ソース | 用途 |
|--------|--------|------|
| 1 | Wikipedia（英語版）/ Transfermarkt | 基本情報、経歴、移籍金 |
| 2 | クラブ公式、監督会見、選手インタビュー | 発言、移籍の背景、クラブ文脈 |
| 3 | UEFA / Premier League などの公式大会サイト | 受賞、記録、代表実績 |
| 4 | 一般紙・大手メディア | 補助的な裏話、補強報道 |

基本方針:
- まず一次情報と公式情報で骨格を作る
- **エピソード、発言、記録、意外な経歴などを積極的に深掘りし、読み応えのある情報量（各項目2〜3文程度）を提供する**
- 推測は書かない
- 単なるWikipediaの要約にならないよう、熱心なファンでも「へぇー」となるようなマニアックな話題や独自の記録を含めること

## `labelled_lines_v1` の書き方

各行は `ラベル::本文` の形式にする。箇条書きではなく、文章としてストーリー性を持たせて記述すること。

よく使うラベル:
- `生まれ`
- `国籍`
- `ポジション`
- `身長・利き足`
- `経歴`
- `移籍金`
- `特徴` （プレースタイルの深掘りを含む）
- `監督・選手の発言`
- `記録`
- `受賞`
- `面白いエピソード`
- `意外な経歴` （生い立ちなどの背景）

## 本文作成ルール

- 最終出力は日本語にする
- `経歴` は 1 カードにまとめる前提で書く
- `経歴` を複数段にしたい場合は、**最初の 1 行だけを `経歴::` にして、以降はラベルなし行で続ける**
- `経歴::` を複数回使ってはいけない。HTML 生成時に別カード扱いになり、エラー対象になる
- 生い立ち、ブレイクしたクラブ、直前所属、現所属が追える構成を優先する
- ユースとトップ昇格が連続する場合は 1 行にまとめてよい
- ポジションは本職を先に書く
- 監督や他の選手のコメントは、時期や文脈が分かるように補足し、選手への評価が伝わるようにする
- `受賞` と `記録` は意味に応じて使い分ける

## セルフレビュー

最低限、以下を確認する。

1. `経歴::` が 2 回以上出ておらず、1 カード内で流れが追えるか
2. ポジション欄が本職よりコンバート先を強く見せていないか
3. 監督・選手の発言欄で別時期のコメントを雑に混ぜていないか
4. ラベル名と本文内容がずれていないか
5. 他選手と比べて 1 人だけ冗長すぎないか
6. **情報が浅すぎないか（単なる事実の羅列になっていないか）**
7. **ファンが喜ぶマニアックなエピソードや独自の記録が含まれているか**

## 出力形式

調査結果は `temp/player_profiles_YYYYMMDD_[team]_[player].md` にまとめる。

推奨構成:

````markdown
# Player Name

```text
生まれ::...
国籍::...
...
```

## Sources

- https://...
- https://...
````

## 良い出力例

```text
生まれ::2002年5月27日、ベルギー・アントワープ
国籍::ベルギー（ガーナ系）
ポジション::右ウィンガー / 左ウィンガー
身長・利き足::173cm / 右利き
経歴::2018-2020: アンデルレヒト（アカデミー→トップチーム）
2020-2023: レンヌ
2023-: マンチェスター・シティ
移籍金::レンヌからシティへ約6500万ユーロ（約5500万ポンド）で加入。
特徴::爆発的なスピードと圧倒的な1対1の突破力を持つ、生粋のドリブラー。相手のハードタックルを避けるために跳躍する独特の避け方を持ち、「The Belgian Flash（ベルギーの閃光）」の異名を持つ。
監督・選手の発言::グアルディオラ監督は「ペナルティエリア内での1対1はアンストッパブル。5〜10ヤードなら彼に敵う者はいない」と絶賛している。また、ベルギー代表の同僚デ・ブライネからは「ダイヤの原石」と称され、シティ移籍前には直接連絡をもらい背中を押された。
記録::EURO 2020のイタリア戦で1試合8回のドリブル成功（10代選手の大会記録）をマーク。レンヌ時代にはリーグ1で1試合15回のドリブル成功を記録し、ネイマールの記録を破った。
面白いエピソード::15歳だった2018年にリバプールへの移籍が迫り、当時のクロップ監督と直接面会もしていた。しかし、当時リバプールに所属していた同郷のサディオ・マネから「まだ若いのだから移籍を急がず、自分の成長を優先すべきだ」と助言を受け、アンデルレヒト残留を決断した。
意外な経歴::ガーナ移民の陸上競技経験者の父を持つ。アントワープにあるスタジアムから車で6分の距離に住み、週末には微かに聞こえる歓声を聞きながら、父や兄とプレミアリーグのハイライトを見て育った。
```
