# 選手プロフィール本文調査スキル

## 概要

この skill は、選手プロフィールの**本文づくり**に特化します。

担当範囲は以下です。

1. 情報収集
2. `labelled_lines_v1` 形式への整形
3. セルフレビュー
4. `temp/player_profiles_YYYYMMDD_[team]_[player].md` 出力

GCS upload、standalone HTML 生成、deploy は親 skill 側で扱います。

## 情報源の優先順位

| 優先度 | ソース | 用途 |
|--------|--------|------|
| 1 | Wikipedia（英語版）/ Transfermarkt | 基本情報、経歴、移籍金 |
| 2 | クラブ公式、監督会見、選手インタビュー | 発言、移籍の背景、クラブ文脈 |
| 3 | UEFA / Premier League などの公式大会サイト | 受賞、記録、代表実績 |
| 4 | 一般紙・大手メディア | 補助的な裏話、補強報道 |

基本方針:
- まず一次情報と公式情報で骨格を作る
- **エピソード、発言、記録、意外な経歴などを積極的に深掘りし、読み応えのある情報量（各項目2〜3文程度）を提供する**
- 推測は書かない
- 単なるWikipediaの要約にならないよう、熱心なファンでも「へぇー」となるようなマニアックな話題や独自の記録を含めること

## 調査の最低要件

各選手について、最低でも以下を確認する。

- 基本情報: Transfermarkt / Wikipedia / UEFA / クラブ公式のいずれか
- クラブ公式プロフィールまたはクラブ公式ニュース
- 本人インタビュー、監督コメント、チームメイトコメント、育成年代関係者コメント、現地紙の特集記事のいずれか 1 本以上
- 日本人選手の場合は、日本語インタビュー・日本語メディアも必ず検索する
- 若手・移籍済み選手の場合は、移籍元/移籍先クラブ側、移籍発表、将来移籍報道も確認する
- 同姓同名や表記ゆれがある選手は、`player_id`、所属クラブ、背番号、現地語表記で取り違えを防ぐ

禁止:
- Wikipedia / Transfermarkt / 公式プロフィールだけで本文を完成させない
- 出典確認なしにSNS・掲示板・まとめサイト由来の話を事実として書かない
- 「すごい」「注目株」「将来有望」などの一般的な形容だけで `特徴` や `面白いエピソード` を埋めない

## 推奨検索クエリ

各選手について、英語・現地語・日本語を組み合わせて検索する。

- `"{player name}" interview`
- `"{player name}" official profile`
- `"{player name}" Sporting profile`
- `"{player name}" childhood academy`
- `"{player name}" nickname`
- `"{player name}" contract extension quote`
- `"{player name}" transfer fee release clause`
- `"{player name}" coach quote`
- `"{player name}" feature profile`
- `"{player name}" long read`
- `"{player name}" 現地紙 インタビュー`
- `"{player name}" ロングインタビュー`
- `"{player name}" エピソード`

ポルトガル、スペイン、フランス、ドイツなどのクラブでは、現地語の以下も使う。

- `entrevista`
- `perfil`
- `renova contrato`
- `cláusula`
- `formação`
- `academia`
- `alcunha`

## 「へぇ情報」の必須化

各選手につき、以下のうち最低 2 種類を本文に入れる。

- 本人インタビュー由来の発言
- 監督・チームメイト・育成年代関係者の具体的な評価
- 育成年代、家族、故郷、愛称に関するエピソード
- 移籍時の交渉、契約解除条項、クラブとの緊張、特殊な契約事情
- 代表・大会・記録に関する意外性のある事実
- ファンが「その選手らしい」と感じるプレースタイル上の具体例
- 大きな試合での象徴的なプレーや、記憶に残るゴール/ミス/復帰

注意:
- 「若くして注目される」「名門で主力」「スピードがある」だけでは、へぇ情報として数えない
- へぇ情報は、本文内で読者が具体的に思い出せる固有名詞・場面・数字・発言を含める

## ネガティブ・私生活情報の扱い

存命人物のネガティブ情報、素行、処分、私生活に関する話は慎重に扱う。

入れてよい条件:
- 本人発言、クラブ発表、競技団体発表、主要メディア、信頼できる現地紙などで確認できる
- キャリア上の転機、移籍、評価、復帰、プレースタイル理解などと明確に関係する
- 断罪調・嘲笑調ではなく、事実と文脈に限定して書ける

入れない条件:
- SNS、掲示板、まとめ、噂レベルのみ
- 単なるバッシング・嘲笑・私生活暴露になる
- 選手理解に寄与しない
- 出典間で内容が食い違い、確認不能なまま断定する必要がある

書き方:
- 「問題を起こした選手」といった人格評価ではなく、「当時こう報じられ、移籍/復帰/評価にこう関係した」という文脈で書く
- 未確認情報は本文に入れず、必要なら作業メモや最終報告で「確認できなかった」と明示する

## `labelled_lines_v1` の書き方

各行は `ラベル::本文` の形式にする。箇条書きではなく、文章としてストーリー性を持たせて記述すること。

よく使うラベル:
- `生まれ`
- `国籍`
- `ポジション`
- `身長・利き足`
- `経歴`
- `移籍金`
- `特徴` （プレースタイルの深掘りを含む）
- `監督・選手の発言`
- `記録`
- `受賞`
- `面白いエピソード`
- `意外な経歴` （生い立ちなどの背景）

## 本文作成ルール

- 最終出力は日本語にする
- `経歴` は 1 カードにまとめる前提で書く
- `生まれ` / `国籍` / `ポジション` / `身長・利き足` は最終HTMLで `基本情報` 1カードに集約される前提で、表記ゆれを避けて簡潔にそろえる
- `経歴` を複数段にしたい場合は、**最初の 1 行だけを `経歴::` にして、以降はラベルなし行で続ける**
- `経歴::` を複数回使ってはいけない。HTML 生成時に別カード扱いになり、エラー対象になる
- 生い立ち、ブレイクしたクラブ、直前所属、現所属が追える構成を優先する
- ユースとトップ昇格が連続する場合は 1 行にまとめてよい
- ポジションは本職を先に書く
- `国籍` はレポート側の国旗表示に流用しやすい一般的な国名表記を優先する
- 現所属クラブ名はレポートのチーム名・ロゴと対応づけやすい表記を優先する
- 監督や他の選手のコメントは、時期や文脈が分かるように補足し、選手への評価が伝わるようにする
- `受賞` と `記録` は意味に応じて使い分ける

## セルフレビュー

最低限、以下を確認する。

1. `経歴::` が 2 回以上出ておらず、1 カード内で流れが追えるか
2. ポジション欄が本職よりコンバート先を強く見せていないか
3. 監督・選手の発言欄で別時期のコメントを雑に混ぜていないか
4. ラベル名と本文内容がずれていないか
5. 他選手と比べて 1 人だけ冗長すぎないか
6. **情報が浅すぎないか（単なる事実の羅列になっていないか）**
7. **ファンが喜ぶマニアックなエピソードや独自の記録が含まれているか**
8. Wikipedia / Transfermarkt の言い換えだけになっていないか
9. 各選手に「へぇ情報」が最低 2 つあるか
10. 本人または関係者の発言・インタビュー由来の情報が 1 つ以上あるか
11. プレースタイル欄に、その選手固有の動き・役割・戦術文脈があるか
12. 面白いエピソード欄が汎用的な褒め言葉で終わっていないか
13. ネガティブ情報を入れる場合、出典と文脈が十分か

## 品質ゲート

以下を満たすまで `temp/player_profiles_YYYYMMDD_[team]_[player].md` を完成扱いにしない。

- Sources に、基本情報ソースと人物像が分かるソースの両方がある
- 本文に、出典で確認した固有の発言・場面・数字・出来事が複数入っている
- `特徴` が「速い」「強い」「テクニックがある」だけでなく、どの局面で効くかまで説明している
- `面白いエピソード` または `意外な経歴` が、熱心なファンでも読んで得るもののある内容になっている
- 噂やネガティブ情報を入れた場合、信頼できる出典とキャリア上の文脈がある
- 逆に、噂止まりの情報を見つけた場合は本文に入れない

作業完了報告では、各選手について品質ゲートを満たすために確認した代表的な断片語句を1つ以上挙げる。

## 出力形式

調査結果は `temp/player_profiles_YYYYMMDD_[team]_[player].md` にまとめる。

推奨構成:

````markdown
# Player Name

```text
生まれ::...
国籍::...
...
```

## Sources

- https://...
- https://...
````

## 良い出力例

```text
生まれ::2002年5月27日、ベルギー・アントワープ
国籍::ベルギー（ガーナ系）
ポジション::右ウィンガー / 左ウィンガー
身長・利き足::173cm / 右利き
経歴::2018-2020: アンデルレヒト（アカデミー→トップチーム）
2020-2023: レンヌ
2023-: マンチェスター・シティ
移籍金::レンヌからシティへ約6500万ユーロ（約5500万ポンド）で加入。
特徴::爆発的なスピードと圧倒的な1対1の突破力を持つ、生粋のドリブラー。相手のハードタックルを避けるために跳躍する独特の避け方を持ち、「The Belgian Flash（ベルギーの閃光）」の異名を持つ。
監督・選手の発言::グアルディオラ監督は「ペナルティエリア内での1対1はアンストッパブル。5〜10ヤードなら彼に敵う者はいない」と絶賛している。また、ベルギー代表の同僚デ・ブライネからは「ダイヤの原石」と称され、シティ移籍前には直接連絡をもらい背中を押された。
記録::EURO 2020のイタリア戦で1試合8回のドリブル成功（10代選手の大会記録）をマーク。レンヌ時代にはリーグ1で1試合15回のドリブル成功を記録し、ネイマールの記録を破った。
面白いエピソード::15歳だった2018年にリバプールへの移籍が迫り、当時のクロップ監督と直接面会もしていた。しかし、当時リバプールに所属していた同郷のサディオ・マネから「まだ若いのだから移籍を急がず、自分の成長を優先すべきだ」と助言を受け、アンデルレヒト残留を決断した。
意外な経歴::ガーナ移民の陸上競技経験者の父を持つ。アントワープにあるスタジアムから車で6分の距離に住み、週末には微かに聞こえる歓声を聞きながら、父や兄とプレミアリーグのハイライトを見て育った。
```
