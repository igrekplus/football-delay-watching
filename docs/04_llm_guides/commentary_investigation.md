# LLM向け指示書: プレミアリーグ実況・解説者調査

> [!NOTE]
> このドキュメントは、AIアシスタントがプレミアリーグ（特にU-NEXT配信）の実況・解説者情報を調査する際の指示書です。

## 1. 目的

ユーザーから「〇〇の試合の実況・解説者を調べて」と依頼された際に、正確かつ効率的に情報を取得するための手順とノウハウを定義する。

---

## 2. 調査フロー

### Step 1: 依頼内容の確認

以下を明確にする：
- **対象試合**: チーム名、日付、節（ラウンド）
- **配信プラットフォーム**: U-NEXT、DAZN、SPOTVなど
- **必要な情報**: 実況者、解説者、ゲスト解説など

### Step 2: 情報ソースへのアクセス（優先順）

| 優先度 | ソース | 方法 |
|--------|--------|------|
| 1 | **U-NEXTフットボール公式X** (@UNEXT_football) | 試合1〜2日前に投稿される「実況・解説ラインナップ」画像を確認 |
| 2 | **PR TIMES** | `site:prtimes.jp U-NEXT プレミアリーグ 解説` で検索 |
| 3 | **U-NEXT公式サイト** | ブラウザツールでログイン後、配信予定ページを確認 |

### Step 3: 情報の抽出

- **画像からの抽出**: 実況解説リストは画像（表）で投稿されることが多い。`capture_browser_screenshot` を活用してスクリーンショットを取得し、内容を読み取る
- **リプライツリーの確認**: スレッド形式で詳細が補足されている場合がある

### Step 4: 結果の報告

以下の形式で報告する：
```
## 実況・解説者情報

| 項目 | 内容 |
|------|------|
| 試合 | [ホーム] vs [アウェイ] |
| 日時 | YYYY/MM/DD HH:MM JST |
| 実況 | [実況者名] |
| 解説 | [解説者名] |
| 情報元 | [URL] |
| 取得日時 | YYYY/MM/DD HH:MM |
```

---

## 3. よくあるエラーと対策

### 3.1 シーズンの混同

**問題**: 年末年始の調査で、前年度の検索結果を今年と誤認する。

**対策**:
1. 検索クエリに **「2025-26」** や **「2025年」** を必ず含める
2. 取得した情報の **投稿日時・記事公開日** を必ず確認する
3. 対戦カードが **今年のカレンダー（第18節など）** と一致するかダブルチェック

### 3.2 日本語実況なし

**問題**: 一部の試合は英語実況のみで日本語解説がない。

**対策**: リストにない試合は「日本語中継なし」の可能性を報告する。

---

## 4. 検索クエリ例

```
# U-NEXT公式Xでの検索
site:x.com/UNEXT_football 実況 解説 2025

# PR TIMESでの検索
site:prtimes.jp U-NEXT プレミアリーグ 解説 2025-26

# 特定試合の検索
U-NEXT マンチェスター・シティ リバプール 解説 2025
```

---

## 5. 注意事項

- **情報解禁タイミング**: 実況・解説者情報は試合の1〜2日前に公開されることが多い。それ以前の調査では「情報未公開」と報告する
- **他プラットフォーム**: DAZN、SPOTVなどU-NEXT以外のプラットフォームでは、情報解禁タイミングやソースが異なる。本指示書を参考に適宜読み替えること
