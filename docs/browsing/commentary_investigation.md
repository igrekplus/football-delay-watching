# プレミアリーグ実況・解説者調査ノウハウ (U-NEXT編)

このドキュメントは、AIアシスタントがプレミアリーグ（特に日本国内独占配信のU-NEXT）における実況・解説者を正確に、かつ効率的に調査するためのノウハウをまとめたものです。

> [!IMPORTANT]
> このノウハウは**プレミアリーグ（U-NEXT）**の調査に基づいています。他のリーグや配信プラットフォームでは情報解禁タイミングやソースが異なるため、適宜読み替えが必要です。

## 1. シーズンの混同防止
年末年始の調査において、最も発生しやすいエラーは「前年度（昨シーズン）の検索結果を今年のものと誤認すること」です。

- **理由**: 年末年始の「スペシャル解説（現役選手・監督の起用）」などのキーワードは、過去のプレスリリースが上位に来やすく、日付（12/26など）が一致するため誤認を招きます。
- **対策**:
  - 検索クエリに必ず**「2025-26」**や**「2025年」**といった西暦とシーズン表記を含める。
  - 取得した情報のソース元の日付（投稿日時や記事公開日）を必ず確認する。
  - 対戦カードが今年のカレンダー（第18節など）と一致しているかダブルチェックする。

## 2. 推奨される情報ソース
U-NEXTの場合、以下の順序で確認するのが最も確実です。

1. **U-NEXTフットボール公式X (@UNEXT_football)**:
   - 試合の1〜2日前に「実況・解説ラインナップ」の画像が投稿される。
   - スレッド形式で詳細が補足されることもあるため、リプライツリーも確認する。
2. **PR TIMES (プレスリリース)**:
   - 「スペシャルウィーク」などの大型企画がある場合、数日〜1週間前に詳細な担当表が公開される。
3. **U-NEXT 公式サイト / アプリの配信予定**:
   - ログインが必要な場合があるため、ブラウザツールでDOMを読み取る際は注意が必要。

## 3. ブラウジングのテクニック
- **X (Twitter) の画像解析**:
  - 実況解説リストはテキストではなく「画像（表）」で投稿されるケースが多いため、`capture_browser_screenshot` を活用してリストを視認・抽出する。
- **検索の絞り込み**:
  - `site:prtimes.jp` や `site:x.com/UNEXT_football` などのコマンドを使用して、公式性の高いソースに絞り込む。

## 4. 特記事項
- 日本語実況・解説が付かない試合（英語実況のみ）も存在するため、リストにない試合は「日本語中継なし」の可能性があることを念頭に置く。
